How to Install and Run

  1. Download TCBS and VENV below

  2. Unzip both TCBS-master.zip and venv-master.zip.

  3. Rename the venv-master folder to venv and drag it into the data folder. The data folder could be found inside of the TCBS-master folder.

  4. If you are using Mac, run TCBS-Launcher by double-clicking it. TCBS-Launcher could be found inside the TCBS-master folder.

    • NOTE: You may need admin permission to run TCBS-Launcher. If you can't get admin permission, or if TCBS-Launcher isn't working, just follow the instructions for running TCBS on Linux below.

  5. If you are using Linux, open a terminal window and change the directory to the data folder inside of the TCBS-master folder.

    cd ~/Downloads/TCBS-master/data

    Then, run __main__.py using venv/bin/python.

    venv/bin/python __main__.py
